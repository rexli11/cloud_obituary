# 楊子佛教禮儀雲端訃聞系統

## 系統簡介
楊子佛教禮儀雲端訃聞系統是一個專門為殯葬服務設計的網路應用程式。系統提供訃聞製作、案件管理、職員管理等功能。

## 主要功能
- 用戶認證與權限管理
- 訃聞製作與預覽
- 案件管理
- 職員管理
- 花禮購物車

## 技術規格
- Python 3.9+
- Django 4.2.16
- MySQL 8.0+
- Bootstrap 5.1.3

## 安裝步驟
1. 克隆專案
2. 安裝依賴套件：`pip install -r requirements.txt`
3. 設定資料庫連線
4. 執行資料庫遷移：`python manage.py migrate`
5. 啟動開發伺服器：`python manage.py runserver`

## 系統需求
- 作業系統：Windows/Linux/MacOS
- Python 3.9 或以上版本
- MySQL 8.0 或以上版本
- 網頁瀏覽器：支援 HTML5 和 CSS3

## 資料庫設定
1. 創建 MySQL 資料庫
2. 在 settings.py 中配置資料庫連線資訊
3. 執行資料庫遷移

## 使用說明
1. 管理員登入系統
2. 新增職員帳號
3. 設定職員權限
4. 開始使用系統功能

## 權限等級說明
- 0：系統管理員
- 1：管理處職權
- 2：案件登錄職權
- 3：訃聞製作職權
- 4：一般職權
